Running JS C&C locally:

1. Go to 
http://www.macromedia.com/support/documentation/en/flashplayer/help/settings_manager04.html

2. Under the drop box "Edit locations...", click "Add location..."

3. Paste in the URL that you get when you run JSCC from your HD or click "Browse for files..." if you know where to find it.

4. Click "Confirm".

5. Reload JS C&C and you ought to have sound!
